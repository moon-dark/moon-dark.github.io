**把浏览器当编辑器**
```javascript
// 浏览器地址栏输入以下地址后，浏览器变成了一个原始而简单的编辑器
data:text/html,<html contenteditable>
// 推而广之，将以下代码放到console执行后，整个页面将变得可编辑
document.body.contentEditable='true';
```
